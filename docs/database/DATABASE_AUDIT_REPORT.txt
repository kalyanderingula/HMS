# CHANGES APPLIED

---

## 1. Primary Key Standardization (patient.sql)

### Changed

| Table | Old PK Type | New PK Type |
|---|---|---|
| genders | BIGSERIAL | UUID |
| blood_groups | BIGSERIAL | UUID |
| marital_statuses | BIGSERIAL | UUID |
| patient_statuses | BIGSERIAL (status_id) | UUID (patient_status_id) |

### Impact

* patients table FK columns changed from BIGINT to UUID
* Column `status_id` renamed to `patient_status_id` for clarity
* Added `created_at` to all master tables

---

## 2. Primary Key Standardization (doctor.sql)

### Changed

| Table | Old PK Type | New PK Type |
|---|---|---|
| doctor_statuses | BIGSERIAL (status_id) | UUID (doctor_status_id) |

### Impact

* doctors table FK changed from `status_id BIGINT` to `doctor_status_id UUID`
* `gender_id` changed from BIGINT to UUID (to match patient module)

# CATEGORY 23 — MULTI-HOSPITAL / MULTI-TENANT MANAGEMENT SYSTEM

## Module Overview

The Multi-Hospital / Multi-Tenant Management System enables a single HMS platform to manage multiple hospitals, clinics, diagnostic centers, and healthcare groups under one enterprise architecture.

This module supports:

* Healthcare Groups
* Multiple Hospitals
* Multiple Branches
* Multi-Tenant SaaS Architecture
* Centralized Governance
* Shared Clinical Services
* Cross-Hospital Patient Management
* Group Financial Consolidation
* Enterprise Analytics

---

# 1. HEALTHCARE GROUP MANAGEMENT

## TABLE: healthcare_groups

### Purpose

Stores healthcare organizations that own multiple hospitals.

### Examples

* Apollo Hospitals Group
* Narayana Health
* Manipal Hospitals
* Aster Healthcare

| Column              | Data Type    | Description  |
| ------------------- | ------------ | ------------ |
| healthcare_group_id | UUID         | Primary Key  |
| group_name          | VARCHAR(255) | Group name   |
| group_code          | VARCHAR(100) | Unique code  |
| created_at          | TIMESTAMP    | Created date |

---

## TABLE: healthcare_group_settings

### Purpose

Stores organization-wide configurations.

### Examples

* Billing Rules
* Branding Settings
* Security Policies

| Column              | Data Type    | Description     |
| ------------------- | ------------ | --------------- |
| setting_id          | UUID         | Primary Key     |
| healthcare_group_id | UUID         | Group reference |
| setting_name        | VARCHAR(255) | Setting name    |
| setting_value       | TEXT         | Setting value   |

---

## TABLE: healthcare_group_brands

### Purpose

Stores hospital brands under the group.

### Example

Apollo Hospitals
Apollo Clinics
Apollo Diagnostics

| Column              | Data Type    | Description |
| ------------------- | ------------ | ----------- |
| brand_id            | UUID         | Primary Key |
| healthcare_group_id | UUID         | Group       |
| brand_name          | VARCHAR(255) | Brand name  |

---

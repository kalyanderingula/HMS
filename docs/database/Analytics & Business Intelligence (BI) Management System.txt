# CATEGORY 21 — ANALYTICS & BUSINESS INTELLIGENCE (BI) MANAGEMENT SYSTEM

## Module Overview

The Analytics & Business Intelligence (BI) System is the centralized reporting, analytics, KPI monitoring, forecasting, and executive decision-support platform of the Hospital Management System.

This module provides:

* Enterprise Reporting
* Executive Dashboards
* KPI Monitoring
* Clinical Analytics
* Financial Analytics
* Operational Analytics
* Predictive Analytics
* Benchmarking
* Data Warehouse Integration
* Compliance Reporting

---

# 1. DATA WAREHOUSE & ETL MANAGEMENT

## TABLE: data_sources

### Purpose

Stores all source systems that feed analytical data.

### Examples

* EMR/EHR
* Billing
* Laboratory
* Pharmacy
* HR

| Column         | Data Type    | Description  |
| -------------- | ------------ | ------------ |
| data_source_id | UUID         | Primary Key  |
| source_name    | VARCHAR(255) | Source name  |
| source_type    | VARCHAR(100) | Source type  |
| created_at     | TIMESTAMP    | Created date |

---

## TABLE: data_warehouse_jobs

### Purpose

Tracks warehouse data loading jobs.

### Examples

* Nightly ETL
* Incremental Sync
* Historical Data Load

| Column           | Data Type    | Description   |
| ---------------- | ------------ | ------------- |
| warehouse_job_id | UUID         | Primary Key   |
| job_name         | VARCHAR(255) | Job name      |
| job_status       | VARCHAR(100) | Status        |
| created_at       | TIMESTAMP    | Creation date |

---

## TABLE: data_marts

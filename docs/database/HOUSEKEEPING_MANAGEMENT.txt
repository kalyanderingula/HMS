# Enterprise Hospital Management System (HMS)

# Housekeeping Management Module Documentation

---

# 1. Module Overview

The Housekeeping Management module tracks cleaning tasks, schedules, inspections, and linen management across the hospital.

This module manages:

* Task type definitions
* Zone-based cleaning assignments
* Task request and completion tracking
* Recurring schedules
* Staff assignments
* Cleanliness inspections
* Linen tracking (issue/return)
* Bed turnover coordination
* Audit trail

This module integrates with:

* Admission & Bed Management (bed turnover triggers)
* Department Management
* HR (housekeeping staff)
* Infection Control

# Enterprise Hospital Management System (HMS)

# Department Management Module Documentation

---

# 1. Module Overview

The Department Management module is the organizational backbone of the hospital management system.

This module manages:

* Hospital departments
* Department hierarchy
* Department units
* Department staff assignments
* Department services
* Operating schedules
* Department workflows
* Cost centers
* Policies
* Performance metrics
* Audit tracking

This module integrates with:

* Doctor Management
* Employee Management
* Appointment System
* Admissions
* Laboratory
* Radiology
* Billing
* Inventory
* HR & Payroll

---

# 2. Enterprise Architecture Principles

This schema follows enterprise-grade standards:

* UUID-based primary keys
* Fully normalized schema
* Parent-child hierarchy support
* Multi-location support
* Audit logging
* Version tracking
* Multi-tenant architecture
* Workflow-driven operations
* Financial cost center support
* Scalable indexing

# 4. Core Department Tables

---

# TABLE: departments

## Purpose

This is the root table of the Department Management module.

## Why This Table Exists

Stores core department information.

## Relationships

* One department can have multiple units.
* One department can have multiple staff members.
* One department can offer multiple services.
* One department can have multiple rooms and equipment.

---

| Column Name        | Data Type    | Description                 |
| ------------------ | ------------ | --------------------------- |
| department_id      | UUID         | Primary key                 |
| tenant_id          | UUID         | Multi-tenant identifier     |
| department_code    | VARCHAR(50)  | Unique department code      |
| department_name    | VARCHAR(255) | Department name             |
| department_type_id | UUID         | FK to department_types      |
| category_id        | UUID         | FK to department_categories |
| status_id          | UUID         | FK to department_statuses   |
| description        | TEXT         | Department description      |
| email              | VARCHAR(255) | Department email            |
| phone              | VARCHAR(20)  | Department phone            |
| established_date   | DATE         | Department start date       |
| floor_number       | VARCHAR(50)  | Floor location              |
| building_name      | VARCHAR(255) | Building name               |
| created_at         | TIMESTAMP    | Creation timestamp          |
| updated_at         | TIMESTAMP    | Update timestamp            |
| created_by         | UUID         | User who created record     |
| updated_by         | UUID         | User who updated record     |
| deleted_at         | TIMESTAMP    | Soft delete timestamp       |
